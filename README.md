## BOFIT

복잡한 보험, 이제 내가 직접 이해하고 선택해요. 보핏과 함께 !

- 개인 맞춤 추천 : 꼭 필요한 보장 중심으로 객관적인 보험 추천
- 쉬운 이해를 위한 리포트 : 복잡한 보험 정보를 시각화하고 친절한 설명을 담은 리포트 제공
- 순수 정보 공유 커뮤니티 : 영업활동이 제한된 공간에서 신뢰할 수 있는 정보 교환
